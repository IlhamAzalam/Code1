Comment héberger un site web sur GitHub

Étapes :

  1. Faites un compte GitHub si vous n'en avez pas déjà un.
  2. Créez un nouveau dépôt en cliquant sur "New" dans votre profil GitHub.
  3. Nommez le dépôt avec la syntaxe suivante : nom-utilisateur.github.io.
  4. Ajoutez vos fichiers de site web dans ce dépôt.
  5. Assurez-vous que votre page d'accueil est nommée index.html.
  6. Enregistrez les modifications et votre site sera accessible à l'adresse : https://nom-utilisateur.github.io.

Comment héberger votre site web sur GitHub :

Vous souhaitez héberger votre site web sur GitHub ? Suivez ces étapes simples :

  1. Créez un compte GitHub si vous n'en avez pas déjà un.
  2. Connectez-vous à GitHub et accédez à votre tableau de bord.
  3. Cliquez sur le bouton "New" pour créer un nouveau dépôt.
  4. Nommez votre dépôt selon la syntaxe suivante : nom-utilisateur.github.io.
  5. Ajoutez vos fichiers de site web à ce dépôt.
  6. Assurez-vous que votre page d'accueil est nommée index.html.
  7. Enregistrez les modifications et votre site sera disponible à l'adresse : https://nom-utilisateur.github.io.
